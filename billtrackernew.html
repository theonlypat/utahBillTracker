Utah State Legislative Bill Search

A Project of ABU

Please use the search function below to find current (and past) bills in the Utah State Legislature. If you know the "Bill ID" of a specific bill (i.e. HB 363), you can use the Search by Bill ID function and enter the Bill ID as well as the Session year to find detailed information pertaining to status of that bill. Otherwise, you can use the Keyword Search and enter keywords to discover more about the bills being proposed at the legislature. Additionally, for a full list of bills (from 2013-2011), leave the search field blank and hit the Search button.
